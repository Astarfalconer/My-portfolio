THE PROPHECY BOT

The prophecy Bot is a simple game that creates random messages composed from 3 data sources to produce a random message. This project has some sentimentality to me as it was my first time experimenting with the fundamentals of JavaScript. By it’s nature of being a project designed to test basic programming skills it uses simple concepts such as Conditionals, switch case statements etc.

Galactic-Playlist

The Galactic-Playlist is a single page react-app making use of Spotify API. This app is able to use it’s corresponding API to search for songs on the Spotify app by name and genre, create a new play-list and then save said play-list on to a user’s Spotify account. This app uses function components instead.
